Seletores CSS

No CSS temos os seletores po ID

  • Seletor por Tipo:
    Busca elementos por uma tag HTML específica
  • Seletor por ID: Busca elementos através do atributo "id"
  • Seletor por Classe: Busca elementos através do atributo "class"
  • Seletor Universal: * Seleciona todos os elementos HTML
  • Seletor de Atributo: Seleciona elementos que possuem um atributo especifico em um documento HTML e conseguimos também buscar atributos com um valor especifico

Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes.Para isso. podemos separar os seletores por virgula(,).

Assim, não precisamos ficar copiando e colando os mesmos códigos de estilo, podemos deixar um bloco só de CSS e reutilizar as mesmas regras.
