Fontes

Existem tipos de fontes que são:

  • Serif (traços e prolongamentos no final dos seus carecteres)
  • Sans-Serif(sem traços no final dos seus carecteres)
  • Diplay(Emfeitada ou comemorativa)
  • Handwriting(Manuescrita)
  • Monospace(Simetreica, suas letras ocupam mesmo espaço)
