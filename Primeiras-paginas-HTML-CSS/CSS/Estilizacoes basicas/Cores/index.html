Definir cores

Existem diversas formas de definimos as cores para utilizarmos no CSS.Que podem ser definida:

  • Por nomes pré-definidos
  • Pela palavra-chave "currentcolor"
  • Pelo código hexadecimal
  • Pelo código hexadecimal mais a tranparência
  • Pelo código RGB, a partir das cores primárias
  • Pelo código RGBA
  • Pelo código HSL
  • Pelo código HSLA

HSL

hsl(hue, saturation, lightness) hsl(hue, saturation, lightness, transparency)
  • Hue (matriz): é o grau na roda de cores( de 0 a 360 graus)
    • 0° ou 360° é vermelho
    • 120° é verde
    • 240° é azul
  • Saturation (saturação): valor percentual
    • 0% um tom de cinza
    • 100% cor total
  • Lightness (luminosideda): luminosideda da cor
    • 0% é preto
    • 100% é branco
